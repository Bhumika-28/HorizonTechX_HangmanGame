# HorizonTechX_HangmanGame

## Description
A simple text-based Hangman game developed in Python where players guess a hidden word one letter at a time.

## Features
- 5 predefined words
- Maximum 6 incorrect guesses
- Console-based gameplay
- User-friendly interface

## Technologies Used
- Python 3

## Concepts Used
- Functions
- Loops
- Conditional Statements
- Lists
- Random Module
- User Input and Output

## How to Play
1. Run the program.
2. Guess one letter at a time.
3. You have 6 incorrect attempts.
4. Guess the complete word before attempts run out.
